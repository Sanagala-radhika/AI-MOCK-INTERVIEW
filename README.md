# 🤖 AI Powered Mock Interview Platform

An AI-powered web application that simulates real-time technical and HR interviews. The platform generates intelligent interview questions, evaluates candidate responses, and provides instant AI-powered feedback to help users improve their interview skills.

## 🚀 Features

- AI-generated Technical and HR interview questions
- Real-time mock interview experience
- Instant AI feedback on answers
- Performance score and analytics
- User authentication
- Responsive and user-friendly interface

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### AI
- Google Gemini API

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sanagala-radhika/AI-MOCK-INTERVIEW.git
```

### Install Frontend

```bash
cd client
npm install
npm start
```

### Install Backend

```bash
cd server
npm install
npm run dev
```
